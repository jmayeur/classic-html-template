# ClassicHTML: Super Simple Static HTML Template

This project is a demonstration of a very basic, retro-style static website using only HTML and CSS—no JavaScript, frameworks, or build tools. The look and feel are inspired by early web browsers like Netscape.

## How to Use

1. **Copy the Template:**
   - Use the file `src/docs/_base_template_.html` as your starting point.
   - Make a copy and rename it for your new page (e.g., `my-article.html`).

2. **Paste Your Content:**
   - Replace the placeholder comment inside the `<div class="retro-center">` with your own content.
   - Use semantic HTML5 elements (e.g., `<article>`, `<section>`, `<header>`, `<footer>`, etc.) for best results.

3. **Format with an Agent:**
   - If you want help formatting your content as valid HTML, paste your draft into the template and ask your agent of choice (such as GitHub Copilot, ChatGPT, or another LLM-based assistant) to format it using valid HTML.

4. **Link Your Page:**
   - Add a link to your new page in `src/index.html` so it appears in the Table of Contents.

## Features

- **Retro CSS:** Simple styles for a classic web look.
- **No JavaScript:** 100% static, works in any browser.
- **Semantic HTML5:** Encouraged for accessibility and clarity.
- **Easy to Extend:** Just copy the template and add your content.
